# Resume-Ranker-AI  

Resume Ranking AI (Django + OpenAI) — Recruiter-focused web app to upload multiple PDF/DOCX resumes, match them against a job description, and generate ranked results (0–100 score) with extracted skills/experience, reasoning, and candidate improvement suggestions.


Key Features:
Multi-resume upload (PDF/DOCX) + job description selection
Automated text extraction and structured parsing (skills, experience, project categories)
Resume–JD match scoring (0–100) with score breakdown and missing-skill insights
Ranked dashboard table + detailed result view
OpenAI-powered reasoning and candidate improvement suggestions (optional)
SQLite development setup with production-ready architecture upgrade path (PostgreSQL/S3/Redis)
